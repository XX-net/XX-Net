XX-Net
========

* Integrate GoAgent with auto search google ip
* Web UI
* Include public appid
* Auto update

Support platform
================
* XP, need tcpip.sys patch, like tcp-z
* Win7
* Win8
* Ubuntu, no systray
* Debian
