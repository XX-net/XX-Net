

XX-Net - A Rebirth of Goagent
========
翻墙工具套件 A firewall circumvention toolkit
* GAE proxy, 稳定、易用、快速  
  Stable, ease-of-use and fast proxy based on GAE  
* Web界面，人性化交互  
  Friendly web UI
